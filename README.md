# Ionic3-Angular4_CountriesApi
Ionic3-Angular4 : Gets all countries and information.

To Run locally:

<ul><li>Clone this git.</li>
<li>Run 'npm install'</li>
<li>Run 'ionic serve -l'</li></ul>

On the real device, we have to modify default config.xml to make API calls without error.<br>
Add this preference.<br>
`<preference name="loadUrlTimeoutValue" value="700000" /><p>`
